Config Generator

Generates configuration files for equipment.

Currently supports:

  • New SRX345 single & dual CPE

In progress:

  • New single ports
  • New ASR servers
  • Decommission SRX345 single & dual CPE
Granite Data Loader

Creates a data file to be used with Granite Data Loader.

Based on the generated data file, Granite Data Loader has the ability to do the following in Xng/Granite:

  • Create ports, channels & IP subnets
  • Assign IP addresses to ports
  • Create equipment (routers, terminal servers, etc.)
